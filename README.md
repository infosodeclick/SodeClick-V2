# LoveBoard Production UI

โปรเจกต์ตัวอย่างเว็บหาคู่ + เว็บบอร์ด + ร้านค้ากรอบโปรไฟล์

## ฟีเจอร์ในโปรเจกต์นี้
- หน้าแรก
- สมัครสมาชิก
- ค้นหา/แมตช์
- โปรไฟล์
- แชท
- เว็บบอร์ด
- กระเป๋าเหรียญ
- แพ็กเกจ VIP
- ร้านค้ากรอบรูปโปรไฟล์
- หลังบ้าน Admin
- ความปลอดภัย

## ระบบกรอบรูปโปรไฟล์
- ซื้อกรอบได้หลายแบบ
- ซื้อซ้ำไม่ได้
- ซื้อแล้วตั้งเป็นกรอบปัจจุบันได้
- ดูกรอบที่ซื้อแล้วในหน้าโปรไฟล์ได้
- เปลี่ยนกรอบใช้งานได้อิสระ

## วิธีรัน
```bash
npm install
npm run dev
```

เปิดเบราว์เซอร์ที่
```bash
http://localhost:3000
```

## หมายเหตุ
เวอร์ชันนี้เป็น frontend prototype สำหรับขึ้น GitHub และใช้ต่อยอดได้
ยังไม่ได้เชื่อมระบบ backend / database / auth / payment จริง
